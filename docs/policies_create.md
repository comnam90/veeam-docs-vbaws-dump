In this article

To create a schedule-based backup policy, do the following:

1. [Launch the Add EC2 Policy wizard](add_policy_launch.md).
2. [Specify a backup policy name and description](add_policy_name.md).
3. [Specify a data protection scope](add_policy_scope.md).
4. [Configure backup source settings](add_policy_source_settings.md).
5. [Enable guest processing](add_policy_guest_processing.md).
6. [Configure backup target settings](add_policy_target_settings.md).
7. [Specify a schedule for the backup policy](add_policy_schedule_retention.md).
8. [Enable AWS tags assignment](add_policy_target_settings_tags.md).
9. [Configure automatic retry, health check and notification settings for the backup policy](add_policy_retry_notification.md).
10. [Review estimated cost of the selected EC2 instances](add_policy_cost.md).
11. [Finish working with the wizard](add_policy_finish.md).
