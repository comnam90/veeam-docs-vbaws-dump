In this article

To create an SLA-based backup policy, do the following:

1. [Launch the Add SLA-Based Policy wizard](add_sla_policy_launch.md).
2. [Specify a policy name and description](add_sla_policy_name.md).
3. [Specify a data protection scope](add_sla_policy_scope.md).
4. [Configure backup source settings](add_sla_policy_source_settings.md).
5. [Configure guest processing options](add_sla_policy_guest_processing.md).
6. [Specify protection settings](add_sla_policy_target_settings.md).
7. [Configure snapshot settings](add_sla_policy_snapshot_settings.md).
8. [Configure worker instance settings](add_sla_policy_worker_settings.md).
9. [Specify general settings for the policy](add_sla_policy_retry_notification.md).
10. [Review the estimated cost of protecting the selected EC2 instances](add_sla_policy_cost.md).
11. [Finish working with the wizard](add_sla_policy_finish.md).
