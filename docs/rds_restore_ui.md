In this article

Veeam Backup for AWS offers the following restore options:

* [RDS instance restore](performing_rds_resource_restore.md) — restore an entire DB instance or an Aurora DB cluster from a restore point.
* [Database restore](performing_rds_database_restore.md) — restore specific databases of Microsoft SQL Server and PostgreSQL DB instances.

You can restore RDS resource data to the most recent state or to any available restore point.
