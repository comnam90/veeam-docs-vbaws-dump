In this article

To create a backup policy, do the following:

1. [Launch the Add EFS Policy wizard](add_policy_launch_efs.md).
2. [Specify a backup policy name and description](add_policy_name_efs.md).
3. [Specify a data protection scope](add_policy_scope_efs.md).
4. [Configure backup source settings](add_policy_source_settings_efs.md).
5. [Enable indexing for the processed file systems](add_policy_indexing_efs.md).
6. [Configure backup target settings](add_policy_target_settings_efs.md).
7. [Specify a schedule for the backup policy](add_policy_schedule_retention_efs.md).
8. [Enable AWS tags assignment](add_policy_target_settings_tags_efs.md).
9. [Configure automatic retry settings and notification settings for the backup policy](add_policy_retry_notification_efs.md).
10. [Review estimated cost of the selected EFS file systems](add_policy_cost_efs.md).
11. [Finish working with the wizard](add_policy_finish_efs.md).
