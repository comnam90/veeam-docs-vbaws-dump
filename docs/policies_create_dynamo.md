In this article

To create a backup policy, do the following:

1. [Launch the Add DynamoDB Policy wizard](add_policy_launch_dynamo.md).
2. [Specify a backup policy name and description](add_policy_name_dynamo.md).
3. [Specify a data protection scope](add_policy_scope_dynamo.md).
4. [Configure backup source settings](add_policy_source_settings_dynamo.md).
5. [Configure backup target settings](add_policy_target_settings_dynamo.md).
6. [Specify a schedule for the backup policy](add_policy_schedule_retention_dynamo.md).
7. [Enable AWS tags assignment](add_policy_target_settings_tags_dynamo.md).
8. [Configure automatic retry settings and notification settings for the backup policy](add_policy_retry_notification_dynamo.md).
9. [Review estimated cost of the selected DynamoDB tables](add_policy_cost_dynamo.md).
10. [Finish working with the wizard](add_policy_finish_dynamo.md).
