In this article

By default, backup policies create only backups of processed DynamoDB tables. At the Targets step of the wizard, you can specify the following backup target settings:

* [Specify backup vaults where Veeam Backup for AWS will store DynamoDB backups](add_policy_target_settings_backups_dynamo.md).
* [Instruct Veeam Backup for AWS to copy DynamoDB backups to other AWS Regions](add_policy_target_settings_backup_copy_dynamo.md).
* [Instruct Veeam Backup for AWS to store DynamoDB backups in a cold storage tier](add_policy_target_settings_backup_archive_dynamo.md).
