In this article

In case of a disaster, you can restore an entire EFS file system from an EFS backup or backup copy. Veeam Backup for AWS allows you to restore one or more EFS file systems at a time, to the original location or to a new location.

How to Perform File System Restore

To restore a protected EFS file system, do the following:

1. [Launch the EFS Restore wizard](restore_entire_launch_efs.md).
2. [Select a restore point](restore_entire_point_efs.md).
3. [Specify account settings for restore](restore_entire_account_efs.md).
4. [Choose a restore mode](restore_entire_mode_efs.md).
5. [Enable encryption for the restored file system](restore_entire_encryption_efs.md).
6. [Specify configuration settings](restore_entire_type_efs.md).
7. [Configure network settings](restore_entire_network_efs.md).
8. [Specify a restore reason](restore_entire_reason_efs.md).
9. [Finish working with the wizard](restore_entire_finish_efs.md).
