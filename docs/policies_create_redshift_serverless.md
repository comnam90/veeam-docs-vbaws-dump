In this article

To create a backup policy, do the following:

1. [Launch the Add Redshift Serverless Policy wizard](add_policy_launch_redshift_serverless.md).
2. [Specify a backup policy name and description](add_policy_name_redshift_serverless.md).
3. [Specify a data protection scope](add_policy_scope_redshift_serverless.md).
4. [Configure backup source settings](add_policy_source_settings_redshift_serverless.md).
5. [Specify a schedule for the backup policy](add_policy_schedule_retention_redshift_serverless.md).
6. [Enable AWS tags assignment](add_policy_target_settings_tags_redshift_serverless.md).
7. [Configure automatic retry settings and notification settings for the backup policy](add_policy_retry_notification_redshift_serverless.md).
8. [Finish working with the wizard](add_policy_finish_redshift_serverless.md).
