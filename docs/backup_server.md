In this article

The backup server is a physical or virtual machine (Windows-based or Linux-based) on which Veeam Backup & Replication is installed. It is the core component in the backup infrastructure. For more information, see the Veeam Backup & Replication User Guide, section [Backup Server](https://helpcenter.veeam.com/docs/backup/vsphere/backup_server.html?ver=120).
