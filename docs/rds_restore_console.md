In this article

Veeam Backup & Replication offers the following restore operations:

* [RDS instance restore](restore_rds.md) — start an entire DB instance or Aurora DB cluster from a restore point.
* [Database restore](rds_database_restore.md) — restore databases of Microsoft SQL Server and PostgreSQL DB instances

You can restore RDS resource data to the most recent state or to any available restore point.
