In this article

To help you assign to IAM roles all the permissions required to perform operations with resources within AWS Organizations, Veeam Backup for AWS allows you to create IAM roles templates that you can then use to configure the necessary roles in AWS.

To create an IAM roles template, do the following:

1. [Launch the Create IAM Roles Template wizard](organization_template_add_launch.md).
2. [Specify IAM roles to include into the template](organization_template_add_name.md).
3. [Specify IAM permissions for the Backup and restore IAM role](organization_template_specify_permissions.md).
4. [Finish working with the wizard](organization_template_add_review.md).
