In this article

To upgrade Veeam Plug-In for AWS, do the following:

1. Install the new version of Veeam Plug-In for AWS as described in section [Deploying Plug-In](installing_plugin.md).
2. Upgrade backup appliances from the Veeam Backup & Replication console as described in section [Updating Appliances Using Console](upgrade_appliance_console.md).
