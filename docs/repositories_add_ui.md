| Important |
| If your backup appliance is managed by a Veeam Backup & Replication server and you add a new backup repository using the Veeam Backup for AWS Web UI, Veeam Backup for AWS will not propagate these settings to the Veeam Backup & Replication server automatically. To discover new backup repositories created on the backup appliance, follow the instructions provided in section [Connecting to Existing Repositories](connecting_existing_repository.md). |

To add a backup repository, do the following:

1. [Check prerequisites and limitations](repository_add_limitations.md).
2. [Launch the Add Repository wizard](repository_add_launch.md).
3. [Specify a backup repository name and description](repository_add_name.md).
4. [Configure backup repository settings](repository_add_folder.md).
5. [Enable data encryption for the backup repository](repositories_add_encryption.md).
6. [Specify an S3 interface endpoint](repositories_add_s3endpoint.md).
7. [Finish working with the wizard](repository_add_finish.md).
