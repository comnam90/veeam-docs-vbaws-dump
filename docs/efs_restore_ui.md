In this article

Veeam Backup for AWS offers the following restore options:

* [File system restore](restore_entire_efs_web_ui.md) — restore an entire Amazon EFS file system.
* [File-level recovery](restore_item_perform_efs.md) — recover individual files and folders stored in a file system.

You can restore EFS file system data to the most recent state or to any available restore point.

|  |
| --- |
| Important |
| Veeam Backup for AWS supports restoring EFS file systems only to the same AWS accounts to which the source file system belong. |
