In this article

To restore the configuration database of a backup appliance using the Veeam Backup for AWS Web UI, do the following:

|  |
| --- |
| Important |
| Before you start the restore process, stop all policies that are currently running. |

1. [Launch the Configuration Restore wizard](config_restore_launch.md).
2. [Choose a backup file](config_restore_file.md).
3. [Review the backup file info](config_restore_file_info.md).
4. [Choose restore options](config_restore_options.md).
5. [Track the restore progress](config_restore_progress.md).
6. [View the results of verification steps](config_restore_verification.md).
7. [Finish working with the wizard](config_restore_finish.md).

Page updated 8/20/2025
