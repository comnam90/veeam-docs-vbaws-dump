In this article

By default, backup policies create only cloud-native snapshots of processed instances. At the Targets step of the wizard, you can enable the following additional data protection scenarios:

* [Instruct Veeam Backup for AWS to replicate cloud-native snapshots to other AWS accounts or AWS Regions](add_policy_target_settings_replica.md).
* [Instruct Veeam Backup for AWS to create image-level backups](add_policy_target_settings_backups.md).
