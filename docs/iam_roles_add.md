In this article

To add an IAM role to Veeam Backup for AWS, do the following:

1. [Check prerequisites and limitations.](byb_roles.md#byb)
2. [Launch the Add IAM Role wizard](iam_roles_add_launch.md).
3. [Specify a name and description for the IAM role](iam_roles_add_name.md).
4. [Specify IAM role settings](iam_roles_add_settings.md).
5. [Specify IAM role permissions](iam_roles_specify_permissions.md).
6. [Finish working with the wizard](iam_roles_add_review.md).
