In this article

To add an AWS Organization to Veeam Backup for AWS, do the following:

1. [Launch the Add Organization wizard](organization_add_launch.md).
2. [Specify a name and description for the organization](organization_add_name.md).
3. [Specify IAM roles that will be used to perform operations with the organization resources](organization_add_settings.md).
4. [Specify an organization scope](organization_add_scope.md).
5. [Finish working with the wizard](organization_add_review.md).
