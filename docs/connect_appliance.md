In this article

If you have already [deployed a backup appliance](deploying_appliances.md), you can add the appliance to the backup infrastructure:

1. [Launch the New Veeam Backup for AWS Appliance wizard](connect_appliance_launch.md).
2. [Choose a deployment mode](connect_appliance_mode.md).
3. [Specify an AWS account in which the appliance resides](connect_appliance_account.md).
4. [Choose the appliance that you want to connect to](connect_appliance_instance.md).
5. [Specify the connection type](connect_appliance_connection.md).
6. [Specify a user whose credentials will be used to connect to the appliance](connect_appliance_creds.md).
7. [Configure repository settings](connect_appliance_repo.md).
8. [Wait for the appliance to be added to the backup infrastructure](connect_appliance_apply.md).
9. [Finish working with the wizard](connect_appliance_finish.md).
