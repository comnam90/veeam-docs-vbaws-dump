In this article

To create a backup policy, do the following:

1. [Launch the Add RDS Policy wizard](add_policy_launch_rds.md).
2. [Specify a backup policy name and description](add_policy_name_rds.md).
3. [Specify a data protection scope](add_policy_scope_rds.md).
4. [Configure backup source settings](add_policy_source_settings_rds.md).
5. [Configure backup target settings](add_policy_target_settings_rds.md).
6. [Specify processing settings](add_policy_processing_settings_rds.md).
7. [Specify a schedule for the backup policy](add_policy_schedule_retention_rds.md).
8. [Enable AWS tags assigning](add_policy_target_settings_rds_tags.md).
9. [Configure automatic retry, health check and notification settings for the backup policy](add_policy_retry_notification_rds.md).
10. [Review estimated cost of the selected RDS resources](add_policy_cost_rds.md).
11. [Finish working with the wizard](add_policy_finish_rds.md).
