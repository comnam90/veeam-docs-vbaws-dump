In this article

To configure the VPC Configuration Backup policy settings, do the following:

1. [Launch the VPC Configuration Backup wizard](vpc_policy_launch.md).
2. [Select AWS Regions to protect](vpc_policy_regions.md).
3. [Specify a backup repository to store an additional backup copy](vpc_policy_add_copy.md).
4. [Configure retentions settings for VPC configuration backups](vpc_policy_retention.md).
5. [Specify notification settings for the backup policy](vpc_policy_notification.md).
6. [Finish working with the wizard](vpc_policy_finish.md).
