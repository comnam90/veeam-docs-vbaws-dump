In this article

Veeam Backup for AWS offers the following restore options:

* [Instance restore](restore_entire_perform.md) — restore an entire EC2 instance.
* [Volume restore](restore_volume_perform.md) — restore EBS volumes attached to an EC2 instance.
* [File-level recovery](restore_item_perform.md) — restore individual files and folders of an EC2 instance.

You can restore EC2 instance data to the most recent state or to any available restore point.
