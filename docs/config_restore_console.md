In this article

To restore the configuration database of a backup appliance using the Veeam Backup & Replication console, do the following:

|  |
| --- |
| Important |
| Before you start the restore process, stop all policies that are currently running on the backup appliance. |

1. [Check prerequisites and limitations](config_restore_limitations.md).
2. [Launch the Configuration Restore wizard](restore_config_launch.md).
3. [Choose a backup file](restore_config_backup.md).
4. [Review the backup file info](restore_config_info.md).
5. [Specify a decryption password](restore_config_pswd.md).
6. [Choose restore options](restore_config_data.md).
7. [Specify a user whose credentials will be used to connect to the appliance](restore_config_creds.md).
8. [Wait for the restore process to complete](restore_config_restore.md).
9. [Finish working with the wizard](restore_config_summary.md).
